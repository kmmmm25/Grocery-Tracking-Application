# Grocery-Tracking-Application
This app automatically tracks the expiration dates of food in your kitchen.　保有している食材の管理を簡単化することを目的としたアプリケーションです。


Grocery App (ver1)
賞味期限をFirebaseでリアルタイム管理するiPhoneアプリです。

💡 主な機能
食材の名前と賞味期限の一覧表示（残り3日以内は赤字で警告）
Firebase（Firestore）へのリアルタイムデータ保存・同期
カメラを起動しての画像撮影（機能実装中）
🛠 技術スタック
Swift / SwiftUI
Firebase (Cloud Firestore)
